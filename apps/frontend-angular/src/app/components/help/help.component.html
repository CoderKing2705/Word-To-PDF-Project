Help & Support

Need assistance? We’re here to help. Below you’ll find answers to common questions about uploading, converting, and downloading files.

Frequently Asked Questions

  • Which file types can I upload?
    Currently, we support .doc and .docx files up to 10MB.
  • How long does conversion take?
    Most files convert within a few seconds. Larger files may take a bit longer.
  • Where are my files stored?
    We store your files temporarily for conversion and remove them automatically after processing.
  • I’m getting an error during upload.
    Check your file size and format. If the issue persists, refresh the page and try again.
